# SEARS - Sistema Inteligente de Gestão de Energia e Arbitragem
## Repositório: Gateway & Intelligence (Raspberry Pi)

Este repositório contém o software central do projeto SEARS, desenvolvido no âmbito da Unidade Curricular de Projeto II. 

### Objetivo
O Gateway é responsável por centralizar os dados recebidos dos nós sensores (ESP32), processar a lógica de arbitragem financeira de energia e comunicar com APIs externas (preços OMIE e Meteorologia).

### Estrutura de Pastas
- `/src`: Scripts Python, Broker MQTT e lógica de decisão.
- `/docs`: Documentação do projeto (ConOps, Proposta, Manuais).
- `/hardware`: Esquemas elétricos e diagramas de arquitetura.

### Tecnologias
- **Linguagem:** Python 3.x
- **Protocolo:** MQTT (Mosquitto)
- **Hardware:** Raspberry Pi 4/5
